1以下哪个选项用八进制来表示十进制整数10？

  • A、0x10
  • B、012
  • C、010
  • D、080

答案：B

4byte类型的取值范围是什么？

  • A、-2^8 ~ 2^8
  • B、0 ~ 255
  • C、取决于Java虚拟机的实现
  • D、-128 ~ 127

答案：D

解析：Java中byte是8位有符号整数，取值范围计算公式：-2^(n-1) ~ 2^(n-1)-1（n为位数），即-128 ~ 127；A错误（2^8=256，范围应为-128~127），B是无符号byte范围（Java无无符号基本类型），C错误（基本类型范围固定，与JVM无关）。

21以下程序代码能否通过编译，如果通过编译，运行“java Sample”时会出现什么情况？

public class Sample{ private void test() { System.out.println(6 >> 33); } public static void main(String[] args) { new Sample().test(); } }
  • A、6
  • B、1
  • C、3
  • D、编译时出错：操作符“>>”的右边的操作元不能大于32

答案：C

解析：1. >>是右移运算符，对于int类型（32位），右移位数n的实际有效位是n % 32（因为32位整数移32位相当于没移）；2. 33 % 32 =1，因此6 >> 33 等价于6 >>1；3. 6二进制110，右移1位为11（二进制），即十进制3。

24以下程序代码能否通过编译，如果通过编译，运行“java Sample”时会出现什么情况？

解析：main方法标准格式：public static void main(String[] 参数名)。要求：1. 访问修饰符public（必须）；2. static（必须）；3. 返回值void（必须）；4. 参数是String数组（[]位置可在参数名前/后，参数名任意）。A正确；B正确（String args[]等价于String[] args）；C正确（static和public顺序可换，参数名var合法）；D错误（访问修饰符顺序错误：void不能在public前，且参数是String而非String数组）。

35Java接口中的方法可以使用哪些修饰符？（多选）

  • A、static
  • B、abstract
  • C、protected
  • D、public

  • B、一个接口可以继承多个父接口

38对于以下程序代码，哪些是合法的表达式，并且取值为true？（多选）

  • A、(f==g)
  • B、(f.equals(new Float(0.9F))
  • C、(f.equals(d))
  • D、(f.equals(g))

答案：BD

39关于JavaDoc文档，以下哪些说法正确？（多选）

40构造方法不能被哪些修饰符修饰？（多选）

  • B、static
  • C、public
  • D、abstract

答案：ABD

解析：构造方法修饰符规则：1. 可以用public、protected、默认、private（C可以，排除）；2. 不能用final（A：final修饰方法不能被重写，构造方法不能被重写，无需final）；3. 不能用static（B：static属于类，构造方法是实例创建时调用，属于对象）；4. 不能用abstract（D：abstract方法需子类实现，构造方法不能被实现）。

41以下哪些选项正确定义了二维的int类型的数组？（多选）

  • A、int a[10][10] = new int[][];
  • B、int a[][] = new int [10][10];

  • D、int [][]a = new int[10][10];

42抽象方法不允许被static修饰。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：抽象方法（abstract）需子类实现，属于实例；static方法属于类，无需实例即可调用，无法被实现。因此抽象方法不能用static修饰，编译报错。

43抽象类不允许被final修饰。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：抽象类（abstract）的核心目的是被继承（子类实现抽象方法）；final类不能被继承，两者矛盾，因此抽象类不能用final修饰，编译报错。

44抽象类不允许有静态方法。（判断：正确/错误）

  • A、正确
  • B、错误

答案：B（错误）

解析：抽象类可以有静态方法（static），静态方法属于类，无需实例即可调用，与抽象类是否被继承无关。例如：abstract class A { public static void test() {} } 编译正常。

45抽象类不允许有final方法。（判断：正确/错误）

  • A、正确
  • B、错误

答案：B（错误）

解析：抽象类可以有final方法，final方法不能被重写，但抽象类的核心是“有抽象方法需子类实现”，非抽象方法可设为final。例如：abstract class A { public final void test() {} public abstract void test2(); } 编译正常。

46final类不允许有静态方法。（判断：正确/错误）

  • A、正确
  • B、错误

答案：B（错误）

解析：final类不能被继承，但可以有静态方法（static），静态方法属于类，与是否继承无关。例如：final class A { public static void test() {} } 编译正常，可通过A.test()调用。

47final类不允许有抽象方法。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：抽象方法（abstract）需子类实现，而final类不能被继承，无法有子类实现抽象方法，两者矛盾，因此final类不能有抽象方法，编译报错。

48所有的Java类都是Object类的直接或间接子类。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：Java是单继承体系，Object类是顶层父类，所有类（包括自定义类、系统类如String）要么直接继承Object（class A {} 等价于 class A extends Object {}），要么间接继承（class B extends A {}，A继承Object）。

49Java类只允许有一个直接的父类。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：Java不支持类的多继承（避免菱形问题），一个类只能有一个直接父类，但可以通过接口实现多继承的效果（类实现多个接口）。

50Java类允许实现多个接口。（判断：正确/错误）

  • A、正确
  • B、错误

答案：A（正确）

解析：Java类支持多实现，用逗号分隔多个接口，例如：class A implements B, C {}（B和C是接口），这是Java弥补单继承的重要特性。

51一个Java接口可以实现另一个Java接口。（判断：正确/错误）

  • A、正确
  • B、错误

答案：B（错误）

解析：接口之间的关系是“继承”（extends），不是“实现”（implements）；一个接口可以继承多个父接口（如interface A extends B, C {}），而“实现”是类对接口的操作（class A implements B {}）。
